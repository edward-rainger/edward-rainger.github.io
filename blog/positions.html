UNDERSTAND CSS Positioning: Relative, Fixed and Absolute.

If you've ever wondered how websites position elements like text or images exactly where they want them, CSS positioning is the magic behind it all.
We are going to explore three key types of positioning in CSS: relative, absolute, and fixed.

Relative Positioning

Elements are positioned relative to their normal position in the document flow.

You can use top, right, bottom, and left properties to move the element from its original position. The space occupied by the element in the normal flow is preserved. Other elements are positioned as if the element were still in its original place. It is practical when you want to change an element from its normal position without affecting the layout of other elements.

Absolute Positioning

Elements are positioned relative to the nearest positioned ancestor (an ancestor that is not static). Similar to relative positioning, offsetting uses top, right, bottom, and left properties to specify the exact position. The element is removed from the normal document flow, so it does not affect the position of other elements. Useful when you want to position an element exactly where you want it, relative to a parent container or the document itself.

Fixed Positioning

Elements are positioned relative to the browser window. Top, right, bottom, and left can be used to specify the exact position. Similar to absolute positioning, the element is removed from the normal document flow. It does not affect the position of surrounding elements. Useful when you want an element to stay fixed relative to the viewport, such as a navigation bar or a header that remains visible as the user scrolls.
